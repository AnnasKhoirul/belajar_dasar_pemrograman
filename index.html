MY PROJECT

Aplikasi Perpustakaan

Ini adalah project pertama saya dalam menggunakan php pada saat magang di IZZAWEB semarang saya membuat aplikasi perpustakaan berbasisi web yang memiliki beberapa fitur seperti login, menambahkan buku, update buku, menambah maupun mengurangi stok buku, melihat siapa yang meminjam buku, stok buku, riwayat peminjaman siswa.

Aplikasi ini dibuat masih menggunakan php native.


Aplikasi Petugas Upacara

Aplikasi petugas upacara ini adalah project pertama saya setelah selesai magang, aplikasi ini adalah tugas dari sekolah untuk mengetes sampai mana kemajuan kemampuan siswanya setelah selesai magang dan saya mendapat pilihan aplikasi ini.

Aplikasi ini mempunyai front end dan back end, di front end sendiri untuk tampilan publik untuk melihat siapa yang menjadi petugas upacara di hari senin dan back end nya sendiri untuk menginputkan petugasnya.


Aplikasi Jepara Adventure

Jepara Adventure ini adalah project android pertama saya, Aplikasi ini bertujuan untuk memperkenalkan kota jepara kepada semua orang baik wisatawan dari luar kota maupun dalam kota. Di aplikasi ini kita bisa mencari wisata, budaya, kuliner, wisata religi yang ada di Jepara.

Dengan adanya aplikasi ini wisatawan bisa dengan mudah mencari informasi tentang Jepara. Aplikasi ini juga dibekali dengan fitur gps untuk memudahkan wisatawan dalam mencari lokasi wisata di Jepara.
